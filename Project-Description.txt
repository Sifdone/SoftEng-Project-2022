# Project Description v0.2

Το AutoSupport θα ειναι ενα συστημα το οποιο θα αποτελειται απο 3 εφαρμογες
και σκοπος του θα ειναι η οργανωση και διαχειριση των πελατων ενος
συνεργειου αυτοκινητων καθως και η παρακολουθηση της εξελιξη της επισκευης
του αυτοκινητου απο τον πελατη.
Πιο αναλυτικα η μια θα ειναι desktop εφαρμογη που θα χρησιμοποιείται από το συνεργειο
και θα παρεχει εργαλεια διαχειρισης πελατων, (διαχειριση στοιχειων, ιστορικο επισκευων κλπ).
Η δευτερη θα ειναι mobile εφαρμογη την οποια θα χρησιμοποιούν οι τεχνικοι του συνεργειου
και η μονη της χρηση θα ειναι η ενημερωση του συστηματος για την προοδο της καθε επισκευης.
H τριτη θα ειναι mobile εφαρμογη που θα δινει στον πελατη την δυνατοτητα να παρακολουθει την
πορεια της επισκευης του οχηματος του, να δει το ιστορικο επισκευων να κλεισει ραντεβου
για επισκευη και ταυτοχρονα θα λειτουργει ως μια διοδος επικοινωνιας μεταξυ του πελατη και του συνεργειου.
